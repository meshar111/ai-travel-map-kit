# AI Travel Map Kit

تطبيق عربي بسيط يساعد أي شخص غير تقني على إنشاء حزمة خرائط سفر قابلة للبيع قبل موسم الصيف.

الفكرة: المستخدم يختار الدولة، المدينة، نوع المسافر، مدة الرحلة، والميزانية. التطبيق يولد له:

- اسم منتج رقمي
- سعر مقترح
- خطة رحلة يومية
- روابط بحث جاهزة في Google Maps
- وصف بيع للمنتج
- برومبتات لإنتاج المحتوى والغلاف والإعلان
- خيار طباعة أو حفظ PDF

## التشغيل المحلي

```bash
node server.js
```

ثم افتح:

```text
http://localhost:4180
```

## تشغيل التقييمات داخل الموقع

التطبيق يعمل بدون مفتاح API عبر روابط Google Maps مباشرة. لإظهار التقييمات وعدد المراجعات داخل الموقع:

1. أنشئ Google Maps Platform API Key.
2. فعّل **Maps JavaScript API** و **Places API**.
3. افتح الموقع وضع المفتاح في حقل `Google Places API Key`.
4. اضغط `جلب الأماكن والتقييمات`.

ملاحظة: في نسخة الواجهة فقط، المفتاح يستخدم داخل المتصفح. للإنتاج التجاري الكبير، الأفضل إضافة Backend بسيط لحماية المفتاح ووضع حدود استخدام من Google Cloud.

## نموذج الدخل

لا يوجد ضمان للدخل، لكن النموذج التجاري المقترح:

- بيع 100 نسخة بسعر 10 دولار = 1000 دولار
- أو بيع 25 نسخة مخصصة بسعر 40 دولار = 1000 دولار

قنوات البيع المقترحة: Gumroad، Etsy، X، TikTok، وطلبات تخصيص مباشرة.
